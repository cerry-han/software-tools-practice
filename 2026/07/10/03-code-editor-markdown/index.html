实验三 代码编辑器与 Markdown

实验三主要完成 VS Code 编辑器配置、Markdown 插件安装和 Markdown 文档编写预览。

VS Code 版本验证

修复 PATH 后，在 PowerShell 中执行 code --version，成功输出 VS Code 版本号。

Markdown 插件安装

在 VS Code 扩展面板中安装 Markdown All in One、Markdown Preview Enhanced、Paste Image 和 markdownlint。

Markdown 编辑

创建 test.md，练习标题、强调、列表、代码块和表格等基础语法。

Markdown 预览

使用 VS Code 的 Markdown 预览功能，对照查看源码与渲染效果。

小结

实验三完成了代码编辑器和 Markdown 写作环境配置，为后续实验报告编写和站点内容维护提供了工具基础。
