本项目由 6 人小组共同完成。为了让任务清晰、成果可追踪，小组按照“环境、文档、Linux、远程管理、部署、整合”进行分工。

| 成员 | 角色 | 主要任务 | 产出 |
|---|---|---|---|
| 成员A | 组长/集成负责人 | 统筹进度、检查材料、整合报告 | 总报告、提交包、README |
| 成员B | Web 环境与 Hexo 负责人 | Node.js、npm、Git、Hexo、站点构建 | 实验二/四材料 |
| 成员C | 文档与 VS Code 负责人 | VS Code、Markdown、插件、截图整理 | 实验三材料 |
| 成员D | Linux/虚拟机负责人 | Ubuntu、Linux 命令、环境变量 | 实验五/六材料 |
| 成员E | SSH/远程管理负责人 | SSH 服务、密钥登录、文件传输 | 实验七材料 |
| 成员F | Git/部署负责人 | Git 仓库、GitHub Pages/Nginx 部署 | 实验八材料 |

## 协作方式

- 使用 GitHub 仓库统一管理源码。
- 每次修改后提交清晰的 commit message。
- 截图按实验编号分类保存。
- 最终由组长统一整合 PDF 报告和源码包。
