本项目由第六组 6 名成员共同完成。为了让任务清晰、成果可追踪，小组按照“环境、文档、Linux、远程管理、部署、整合”进行分工。

| 成员 | 角色 | 主要任务 | 产出 |
|---|---|---|---|
| 何银尧 | 组长/集成负责人 | 统筹进度、搭建展示站点、整合报告和最终提交 | 总报告、源码仓库、展示站点 |
| 龚嫣然 | 远程管理负责人 | SSH 服务、密钥认证、SCP/rsync 文件传输、远程 Git 仓库验证 | 实验七材料 |
| 董晗 | Web 环境与站点构建负责人 | Node.js、npm、Git、Hexo 环境配置和站点构建 | 实验二/四材料 |
| 谢娩婷 | 文档与代码编辑器负责人 | VS Code、Markdown 插件、实验文档编辑和预览 | 实验三材料 |
| 任鑫瑜 | Linux/虚拟机负责人 | VirtualBox、Ubuntu Server、双网卡和 Linux 基础配置 | 实验五/六材料 |
| 程奕菲 | 部署与验收材料负责人 | GitHub Actions、GitHub Pages、Nginx/虚拟机部署和验收素材整理 | 实验八材料 |

## 协作方式

- 使用 GitHub 仓库统一管理源码。
- 每次修改后提交清晰的 commit message。
- 截图按实验编号分类保存。
- 最终由组长统一整合 PDF 报告和源码包。
